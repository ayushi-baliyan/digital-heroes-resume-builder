# 📄 Digital Heroes Resume Builder

A modern and responsive Resume Builder built using **React + Vite**. Users can easily enter their details, preview their resume in real time, and download it as a PDF.

## 🌐 Live Demo

🔗 https://digital-heroes-resume-builder.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/ayushi-baliyan/digital-heroes-resume-builder

---

## ✨ Features

- Create professional resumes
- Live resume preview
- Download resume as PDF
- Responsive design for desktop and mobile
- Simple and user-friendly interface
- Fast performance with React + Vite

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ayushi-baliyan/digital-heroes-resume-builder.git
```

Go to project directory

```bash
cd digital-heroes-resume-builder
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
